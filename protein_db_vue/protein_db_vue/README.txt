Para correr o nosso programa é necesario ter a versao 8.9 ou cima de Node.js e uma versao atualizada de npm

	Para fazer download de Node.js
	https://nodejs.org/en/download/

	Para atualizar a versao do nmp:
	npm install -g npm@latest


Necessario instalar VUE CLI e VUE Bootstrap
npm install -g @vue/cli
npm i vue bootstrap-vue bootstrap


Para correr a aplicação entrar dentro da pasta proteindb e correr
npm run serve
